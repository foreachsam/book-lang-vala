pkg 參數的使用

撰寫簡單的Gtk程式

原始檔

int main(string[] args) {
    Gtk.init(ref args);
    var win = new Gtk.Window();
    win.show_all();
    Gtk.main();
    return 0;
}

編譯程式碼

執行下面指令編譯，使用了一個參數「--pkg」，值為「gtk+-3.0」。

$ valac app.vala --pkg gtk+-3.0

會產生一個執行檔「app」。

執行程式

$ ./app
