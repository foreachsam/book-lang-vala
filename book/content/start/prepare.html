Vala 學習筆記

環境準備

安裝 valac

在Ubuntu的環境，只要執行下面的指令，安裝「valac」這個「Metapackage」就行了。

$ sudo apt-get install valac

或是找尋特定的版本

$ apt-cache search valac

例如:安裝「valac-0.22」

$ sudo apt-get install valac-0.22

觀看版本

安裝好後，執行下面的指令觀看版本

vala --version

或是

valac --version

顯示

Vala 0.22.1

觀看 manual

$ man vala

$ man valac
