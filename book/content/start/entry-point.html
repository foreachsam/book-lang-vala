程式進入點

程式進入點，是尋找「main」這個「function」或是「static method」。

基本形式

沒有回傳，沒有參數。

void main() {

}

沒有回傳，有參數。

void main(string[] args) {

}

有回傳，沒有參數。

int main() {
    return 0;
}

有回傳，有參數。

int main(string[] args) {
    return 0;
}

namespace 形式

沒有回傳，沒有參數。

namespace App {
    void main() {

    }
}

沒有回傳，有參數。

namespace App {
    void main(string[] args) {

    }
}

有回傳，沒有參數。

namespace App {
    int main() {
        return 0;
    }
}

有回傳，有參數。

namespace App {
    int main(string[] args) {
        return 0;
    }
}

class 形式

沒有回傳，沒有參數。

class App {
    public static void main() {

    }
}

沒有回傳，有參數。

class App {
    public static void main(string[] args) {

    }
}

有回傳，沒有參數。

class App {
    public static int main() {
        return 0;
    }
}

有回傳，有參數。

class App {
    public static int main(string[] args) {
        return 0;
    }
}
